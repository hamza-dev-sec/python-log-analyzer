# Python Log Analyzer

## Overview
This project is a Python script designed to analyse log files and detect patterns such as repeated IP addresses, errors, and suspicious activity.

The tool helps demonstrate basic log analysis techniques used in cybersecurity and system monitoring.

## Features
- Reads and processes log files
- Detects repeated patterns
- Identifies potential suspicious entries
- Simple and readable Python implementation

## Technologies Used
- Python
- File handling
- Regular expressions (optional depending on implementation)

## Example Use Case
Log analysis is commonly used in cybersecurity to detect unusual behaviour in systems such as:
- repeated login attempts
- suspicious IP addresses
- error patterns in server logs

This project demonstrates how Python can be used to automate this process.

## Future Improvements
- Add support for multiple log formats
- Implement visual output or charts
